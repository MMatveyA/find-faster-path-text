\section*{Введение}
\addcontentsline{toc}{section}{Введение}

\textit{Целью} курсовой работы является демонстрация знаний,
полученных при изучении дисциплины «Информатика». В данной работе
необходимо поставить конкретные задачи по заданной теме, которые будут
определять весь ход выполнения курсовой работы.

\textit{Актуальность} данной темы (разработка программы для нахождения
кратчайшего пути между двумя точками) связана с тем, что поиск и
построение наиболее оптимального пути является важной задачей во
многих сферах: от доставки товаров по всему миру до оптимизации работы
с графами в прикладных
целях\cite{Sokolov-graph_in_logistic,Vardomatskaya-optimization_with_graphs}. Последнее
особено важно, так как графы используются во многих алгоритмах и
повзоляют проще отобразить некоторую структуру в программном
коде~\cite{Брюске1998-графыАлгоритмов}.

Выполнения курсовой работы требует творческого подхода и всестороннего
исследования поставленных задач.

\emph{Основными задачами курсовой работы являются:}
\begin{itemize}
	\item применение знаний и умений работать с языком программирования к
	      конкретно поставленному заданию;
	\item разработка словесного алгоритма программы для обучения
	      нахождению кратчайшего пути между точками в соответствии с
	      поставленными условиями;
	\item реализация данного алгоритма на языке программирования C++,
	      т.е. непосредственная разработка алгоритма в программе C++ с помощью
	      ее синтаксиса.
\end{itemize}

\emph{Основные этапы курсовой работы:}
\begin{enumerate}
	\item Анализ поставленной задача;
	\item Выбор обоснования и изложения метода решения поставленной
	      задачи;
	\item Построение алгоритма решения поставленной задачи;
	\item Создание и отладка программы;
	\item Разработка комплекта тестов для проверки правильности работы
	      программы;
	\item Оформление пояснительной записки;
	\item Защита курсовой работы;
\end{enumerate}

Таким образом, в процессе выполнения работы необходимо научиться
работать с литературой и интернет-ресурсами, список которых включен в
конец курсовой работы.

\section{Постановка задач}

Выполнение задания следует начинать с четкого уяснения поставленной
задачи. Задача курсовой работы звучит следующим образом: найти
кратчайший путь между двумя шариками, расположенными на двух
произвольных клетках в поле 10x10 клеток, при условии, что движение
возможно толькко по вертикали, горизонтали и диагонали. То есть в
начале задаётся две случайные координаты, в которых распологаются
шарики, и надо найти и построить кратчайший путь между ними.

Данная задача относится к своего рода обучающим задачам, так как чтобы
найти кратчайший путь между двумя точками, необходимо обучиться
алгоритму решения такой задачи. Метод решения данной задачи должен
ясно описывать, каким образом получается исходный результат.

Существует несколько методов решения такой задачи. Необходимо выбрать
наиболее оптимальный. Критерии выбора метода решения данной задачи, на
которые следует опираться, следующие:
\begin{enumerate}
	\item Легкость реализации в виде алгоритма;
	\item Время решения задачи;
	\item Возможность проверки правильности решения.
\end{enumerate}

\section{Алгоритм задачи}

Существует несколько способов описания алгоритма. Рассмотрим словесное
описание алгоритма. В словесном описании алгоритм выполнения выглядит
следующим образом:

\begin{enumerate}
	\item С клавиатуры вводятся две пары координаты;
	\item Из координаты точки \verb|point2| соответственно вычитаются
	      координаты точки \verb|point1|, результат заносится в переменную
	      \verb|subtraction_point|;
	\item Если \verb|subtraction_point.x| равен 0 и
	      \verb|subtraction_point.y| равен 0, то обе точки находятся в одной
	      клетке, следовательно решения не существует (получается пустое
	      перемещение).
	\item В противном случае из координат переменной
	      \verb|subtraction_point|, если эта координата не равна нулю,
	      вычитается равная по знаку единица до тех пор, пока точки не
	      совпадут. Дополнительное получающиеся координаты завписываются в
	      массив \verb|result|.
\end{enumerate}

Тот же алгоритм, выполненный в варианте псевдокода.  \IncMargin{1em}
\begin{algorithm}
	\caption{Алгоритм нахождения кратчайшего пути}
	\LinesNumbered{}

	\SetKwData{Left}{left}\SetKwData{This}{this}\SetKwData{Up}{up}
	\SetKwFunction{Union}{Union}\SetKwFunction{FindCompress}{FindCompress}
	\SetKwInOut{Input}{input}\SetKwInOut{Output}{output}

	\Input{Две пары координат} \Output{Массив координта для достижения
		цели} \KwData{$x_1 \geqslant 0$, $x_2\geqslant 0$, $y_1\geqslant 0$, $y_2 \geqslant 0$} \BlankLine

	point1 \gets{($x_1$, $y_{1}$)}\;

	point2 \gets{($x_2$, $y_2$)}\;

	result \gets{CreateArray(0)}\;

	subtractionPoint \gets{$point2 - point1$}\;

	\If{(subtractionPoint.x = 0 and subtractionPoint.y = 0)}{ return
		result\; }

	\While{($subtractionPoint.x \neq 0$ and
		$subtractionPoint.y \neq 0$}{ \If{$subtractionPoint.x \neq 0$}{
			$subtractionPoint.x \gets subtractionPoint.x -
				\frac{subtractionPoint.x}{abs(subtractionPoint.x)}$\;
		}\If{$subtractionPoint.y \neq 0$}{
			$subtractionPoint.y \gets subtractionPoint.y -
				\frac{subtractionPoint.y}{abs(subtractionPoint.y)}$\; }
		$result \gets AppendArray(result, subtractionPoint)$\; }

	\KwResult{result}
\end{algorithm}
